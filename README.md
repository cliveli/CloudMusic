CloudMusic
=================

usage:
   python ClouldMusic.py <type> <keyword> <savepath> </br>
* type:  
   * song  
   * album  
   * playlist  
   * playlistid  
* keyword:  
   * playlist id when type is playlistid  
   * keyword when type is other  
* savepath:  
   * local path to save songs
