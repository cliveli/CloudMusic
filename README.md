CloudMusic
=================

usage:
    python ClouldMusic.py <type> <keyword> <savepath> </br>
    type:</br>
        song</br>
        album</br>
        playlist</br>
        playlistid</br>
    keyword:</br>
        playlist id when type is playlistid</br>
        keyword when type is other</br>
    savepath:</br>
        local path to save songs</br>
